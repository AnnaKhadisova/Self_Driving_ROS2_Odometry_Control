Implementation of robot modeling, odometry estimation, and motion control using ROS2 simulation environments Gazebo.

Implemented Components:

- Robot modeling using URDF
- Robot visualization in RViz and Gazebo
- Odometry estimation
- Coordinate-frame transformations using tf2
- Velocity control
- Navigation and robot-state estimation
  
Technologies:
- ROS2
- Gazebo
- RViz
- URDF
- tf2
- Python
- C++
